## Was sind Subliminals und wie funktionieren sie genau?

**Subliminals** — das ist der Fachbegriff für Audiodateien, die positive Suggestionen (Affirmationen) so leise enthalten, dass dein bewusster Verstand sie nicht wahrnimmt. Die Botschaften sind unterhalb deiner Hörschwelle versteckt, hinter Musik oder Naturgeräuschen maskiert.
